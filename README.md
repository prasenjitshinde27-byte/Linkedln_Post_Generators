LinkedIn Post Generator 🚀
An AI-powered LinkedIn Post Generator built using Python and Google's Gemini AI. This application helps users create engaging, professional, and audience-focused LinkedIn posts in seconds.
 =>FEATURES✨
*Generate LinkedIn posts on any topic
*Multiple writing tones (Professional, Casual, Inspirational, etc.)
*Audience-specific content generation
*Different post length options
*Uses Gemini AI for high-quality content creation
*Save generated posts to text files
*User-friendly command-line interface
